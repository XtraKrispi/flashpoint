## Copyright GMT Games

An attempt at an implementation of the game, all rights belong to GMT Games

The application is built using Elm, Lamdera, and Tailwind.
Prerequisites:
* [Lamdera](https://lamdera.com/)
* [NodeJs/NPM](https://nodejs.org/en) 

To run:

Windows:
    
    npm install

In two terminals:
    
    npm run tailwind
    lamdera live

Unix:

    npm install
    make -j
